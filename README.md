## 🚀 About Me

Hi! I'm **Hatziry Vitales Herrera**, a **Computer Engineering** student at the Escuela Superior de Cómputo - **IPN (ESCOM)**. I am deeply passionate about secure software development, systems architecture, and solving complex problems through clean, efficient code.

My core areas of interest and specialization include:

* **Cryptography & Cybersecurity:** Designing and implementing robust security frameworks, digital signatures (ECDSA), authenticated encryption (AES-GCM), and identity validation.
* **Distributed Systems & Networking:** Software architecture, concurrency management, and scalable solutions.
* **Software Development:** Building resilient applications by combining strong backend logic with functional design.

### 🧶 Beyond the Terminal — Algorithms in Yarn

When I'm not writing code, I'm running algorithms of a different kind: **I crochet**. And honestly, it's not that far from programming. A crochet pattern is essentially pseudocode — sequential instructions, iteration over rows, state tracking (stitch counts, tension), and pattern recognition to spot when something breaks. Debugging a miscounted round is the same mindset as tracing an off-by-one error. It's where I exercise algorithmic thinking, patience, and structured problem-solving away from the screen.
